A simple background-size: 300% with animated background-position. The cheapest approach, works everywhere. Best for subtle two-to-four-stop loops.

A conic-gradient circle with filter: blur and a continuous rotation. GPU-accelerated via will-change: transform. Great for halos and accent orbs.

Shifting Spectrum

CSS background-clip: text with an animated background position. Faster duration (6s) works here because it's a small element, not a full-screen loop.

SVG Gradient

SVG Shape Fill

SVG <animate> on stop-color lets gradients fill any shape — rounded rects, text outlines, logo paths. Not GPU-accelerated, so keep SVGs modest in size.
