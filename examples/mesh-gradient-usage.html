MeshGradient

Drop-in animated mesh gradient for any container. One import, one line to create.

1. Simplest usage — just a palette name

// Import the module import { MeshGradient } from './mesh-gradient.js'; // Attach to any container — that's it const gradient = new MeshGradient('#hero', { palette: 'aurora', });

Hello world

Just set a palette name. Everything else uses sensible defaults.

2. Tuned settings — slow, broad, subtle

new MeshGradient('#hero', { palette: 'brand', speed: 0.03, // barely moving zoom: 0.4, // very broad color washes intensity: 0.25, // subtle, not overpowering octaves: 1, // smooth, no detail layers });

Ambient background

Low speed + low intensity = a gradient that breathes without distracting from content.

3. Turbulent — fast, detailed, vivid

new MeshGradient('#hero', { palette: 'synthwave', speed: 0.12, // energetic zoom: 1.8, // tight swirling patterns intensity: 0.5, // vivid colors octaves: 3, // layered detail });

Plasma energy

High octaves add fractal detail. Combined with high zoom and speed, you get dense, swirling motion.

4. Light mode — white background

new MeshGradient('#hero', { palette: 'brand-light', speed: 0.04, intensity: 0.4, });

Light mode

Light palettes use tint blending instead of additive, so colors stay true on white.

5. Custom palette — bring your own colors

new MeshGradient('#hero', { palette: { bg: [5, 5, 20], colors: [ [255, 50, 80], // red [255, 180, 0], // amber [0, 200, 255], // electric blue [180, 0, 255], // purple ], light: false, // dark bg → additive blend }, speed: 0.07, zoom: 0.8, intensity: 0.45, octaves: 2, });

Custom colors

Pass any RGB arrays. Use light: true for white/light backgrounds, false for dark.

6. Cleanup

// Stop animation and remove the canvas gradient.destroy();

API Reference

OptionTypeDefaultDescription
palettestring | object'brand'Built-in name or custom { bg, colors, light } object
speednumber0.06Animation speed. 0.03 = glacial, 0.06 = gentle, 0.15 = rapid
zoomnumber0.6Noise scale. 0.3 = huge soft washes, 1.0 = medium, 2.5 = tight swirls
intensitynumber0.4Color strength. 0.2 = barely there, 0.4 = balanced, 0.6 = vivid
octaves1 | 2 | 31Noise layers. 1 = smooth, 2 = some texture, 3 = turbulent detail
seednumber42Random seed. Same seed = same pattern across loads
